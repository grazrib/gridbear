What are Skills?

  • User Skills are reusable prompt templates
  • Use them with memos for scheduled execution
  • Or invoke directly: "use skill skill_name"
  • Context Skills are plugin instructions
  • They are injected into the system prompt
  • Edit them to customize how the AI uses each plugin
