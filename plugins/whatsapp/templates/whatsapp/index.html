About WhatsApp

WhatsApp channel via Evolution API (self-hosted).

  • Text messages
  • Voice notes & TTS
  • Image & document sharing
  • Inter-agent communication
  • Memos & reminders

Commands

!help - Show commands

!reset - New session

!stop - Cancel request

!memo - List memos

!memory - Memory mgmt

Setup

1. Set EVOLUTION_API_KEY in .env

2. Add whatsapp channel to agent YAML

3. Start containers with docker-compose up

4. Click "Connect" and scan QR code

5. Send a message to test
